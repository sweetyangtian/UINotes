# Ellipsis
::: tip 
文本溢出，hover文本显示toolTip提示
:::
```jsx 
<Ellipsis placement="topCenter">{text}</Ellipsis>
```
| 参数名        | 说明             | 必需 | 类型 / 默认值 | 备注  |
|------------|----------------|----|----------|-----|
| children   | 提示文字           | 是  |          |     |
| line       | 控制多行溢出         |    | 1        |     |
| ...rest | antd 中 toolTip组件所支持的所有api |
